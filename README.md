# WiDS_MLMeetsSunAndWind_Project_EndSem
A project on mapping solar suitability areas (PV) in Telangana State using Machine Learning and GIS

A Github repository to see and understand my WiDS - ML meets Sun and Wind Project. 
Updated on 07 anuary 2025.
v1.0
-> This Repository contains ipynb files for preprocessing and modelling...
-> layerAlignment.ipynb - shows the code for all input layers alignment for matching to create the predicted map using ML algorithm.
-> LayerEncoding_sun.ipynb - shows the code for layer conding such as normalization, LULC encoding, Aspect encoding.
-> WiDS_Random_Forest.ipynb - Code showing ML model prediction using Random Forest.
-> RF_model_bilinear.ipynb - Code showing ML model prediction using Random Forest (bilinear interoplation inputs).
-> WiDS_XGBoost.ipynb - Code showing ML model accuracy using XGBoost.
-> WiDS_XGBoost_bilinear.ipynb - Code showing ML model accuracy using XGBoost (bilinear interoplation inputs).
-> endSemReport.pdf - Report on the project.
